# VS1 playtest checklist — Tier A

**Build:** `Builds/Win64/` dev release.  
**Tester:** 1–3 người, 3 session × 15 min.

## Functional

- [ ] Boot <15s cold start
- [ ] 3 plots visible
- [ ] Moisture decreases over ~5 min sim (or time scale)
- [ ] Water increases moisture; clamp 100
- [ ] Wilt slows growth; recover after water
- [ ] Harvest only when mature
- [ ] Inventory increments
- [ ] Quit/reopen preserves state

## Feel

- [ ] Hiểu cần tưới không đọc doc
- [ ] Không annoy popup (FTUE ≤5 toasts)
- [ ] FPS stable windowed
